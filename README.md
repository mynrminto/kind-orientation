# kind-orientation

KInd Agents オリエンテーション課題の提出リポジトリです。

## Codex をどう使ったか
- テンプレートを渡して「profile.md を作って。内容は質問しながら埋めて」と依頼
- Codex の質問に答える形で基本情報を記入し、体裁も整えてもらいました
- GitHub への push も Codex に手順を教わりながら実施しました

(動作確認用の更新)

## この課題について

この課題は Codex を使って作成しました。
自己紹介の内容整理と Markdown の清書に Codex を活用しています。
GitHub 上のファイル作成と README の更新も Codex で行いました。
